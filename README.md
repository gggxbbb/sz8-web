# sz8

丐版 base64 编码, 使用 `常熟市中学我爱你` 这 8 个字符 编码一切字符串

## Usage

没有 usage, 只有 [demo](https://sz8.gxb.pub)

现已支持 [自定义字符池](https://sz8.gxb.pub/any.html)
